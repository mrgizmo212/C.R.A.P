What This C.R.A.P. Is

Run C.R.A.P. alongside your app locally to identify bugs, add detailed comments, capture and annotate screenshots, then package everything into a folder Cursor can evaluate, plan from, and use to make corrections.

Local Only
What is a review?

One saved capture session: the site you inspected, what looked wrong, screenshots, notes, annotations, and a generated Cursor handoff prompt.

How do I use it?

Load a site, select an element or box-select a region, annotate screenshots, add notes, then save this C.R.A.P. into a local evidence folder.

What happens after save?

The server writes Markdown, JSON, screenshots, an AI/fallback report, and a prompt telling Cursor exactly where to find the evidence.

  1. Enter a local or reachable site URL and click Lock and Load.
  2. Use Select Element when same-origin inspection works, or Box Select when it does not.
  3. Add notes and annotated screenshots, then click Save This C.R.A.P.
